# Cleanos

## What is cleanos?
Cleanos is an operating system that i'm building from scratch :)

## What's gonna be different from others
The base for all apps that will run on this os will be my interpreted programing language: cleanskript, all the apps that are written in other languages will be ran using a built in speedy translation layer written in cleanskript. Frame interpolation will be system wide enabled to double fps in all apps and situations for extreme smoothness.

## Why is it called cleanos
It sounds good.

## Why use it once it releases
It's gonna run fast and be great :)
