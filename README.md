# Cleanos

## What is cleanos?
Cleanos is an operating system that i'm building from scratch, it's gonna be run on nodejs.

I'm gonna build a custom kernel for the os and compile nodejs for it, a canvas the size of the screen will be controlled by a nodejs script and will make the os interface, every user input will be processed by the nodejs script. That makes it a nodejs os.
